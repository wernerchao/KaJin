我的線上簡歷

2016/03 ~ now

Knowledge:

Inkscape, Jquery, Jade, Scss, D3.js, RWD

Learn:

我的線上簡歷就是現在這份網頁，對我而言是一個突破，因為它是第一個由我自己設計美工，並著重 UI/UX 的前端網頁。以往我所寫的網頁不是對於UI/UX沒有著墨，只是主要把重心放在Javascript的撰寫和結構上。

FireWork

2015/10 ~ now

Knowledge:

Reactjs, Jquery, html5 canvas api, Nodejs, Mongodb, Youtube api, physics

Learn:

會寫這個網頁，是當初看到煙火覺得很酷，但卻沒有專案將它實作在網頁上，因此便嘗試想要讓煙火可以在瀏覽器上放。

這個專案是我做的第二個大型網頁專案，學到了很多東西。一開始憑藉著一些高中物理的知識，很快的就把煙火的 prototype 做出來了，雖然成功做出了煙火，但是卻十分的 lag。於是又花了很多的心力，想要把任何可能造成繪圖效率變慢的因素找出來。在這過程中，我學會了如何改變繪製的順序，大幅的優化繪製的效率；我也學會了使用 javascript prototype，並設計更良好的程式碼，又把同一時間能繪製的煙火數大大的提升。此外，為了使煙火更有真實性，我也加入了空氣阻力等因素。

接著，參酌網友的意見，我也不斷的重構程式碼。一開始本來打算用 Jquery 直接做出這個系統，但是專案進行到一半就崩潰了。這時便導入 Reactjs，成功的加入了煙火回放系統和 Youtube 音樂配樂。

希望你喜歡我的電子煙火！

Link:

https://noootown.github.io/FireWork/
