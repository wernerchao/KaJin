心理師的話

每一個人的生命都像是一本厚厚的書，翻開來我們才知道箇中的故事，

每一個人之所以形成「現在」，都是有大量的「過去」所交織成的。

藉由諮詢，我們才有機會慢慢地找回自己，

更認識自己的每一個樣貌，才能更進一步地解決生命中的問題。

 

而關係，往往是兩個人互動出來的舞步，

兩個人帶著自己的過去，與現在的對方互動著，

唯有瞭解自己、瞭解對方，方能夠跳出優雅的關係之舞。

 

生命沒有絕對的對與錯，

只要我們釐清了自己在生命中經歷了什麼、

在關係中如何與他人交織生命，我們終究能夠重拾生命的興味。
